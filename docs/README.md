# Welcome to DocsDeploy

This is your documentation homepage. DocsDeploy automatically reads from this `docs` folder and creates a beautiful documentation website.

## Getting Started

DocsDeploy is a powerful documentation platform that can read from either:
- Local `docs` folder (like this one)
- GitHub repository

## Features

- 📚 **Auto-generated navigation** from your folder structure
- 🎨 **Beautiful, responsive design** 
- 📝 **Full Markdown support** with syntax highlighting
- 🔗 **GitHub integration** for live documentation
- 📱 **Mobile-friendly** interface

## Quick Start

1. Add your Markdown files to the `docs` folder
2. Organize them in subdirectories as needed
3. The navigation will be automatically generated
4. Start writing great documentation!

## Example Structure

```
docs/
├── README.md (this file)
├── getting-started.md
├── api/
│   ├── authentication.md
│   └── endpoints.md
└── guides/
    ├── installation.md
    └── configuration.md
```

Happy documenting! 🚀
